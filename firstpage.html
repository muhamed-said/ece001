Logic Programming Language

programming languages let programmers make declarative statements and then allow the machine to reason about the consequences of those statements. In a sense, this language doesn’t tell the computer how to do something, but employing restrictions on what it must consider doing. To call these groups ” types of language ” is really a bit confusing. It’s easy to program in an object-oriented style in C language.
